Matching Dice

In order to win, you must match the number that is displayed to match the number of the card you have displayed (the jack card will count as 11).

To start, press the roll dice button and it will generate a random number every few seconds up until 5 turns.

special numbers

The following two numbers hold special meaning to this game.

  • 2: If the game generate a number 2, you'll get an extra turn regardless of which turn you're on at the moment.

  • 12: If the game generate a number 12, it will end the game and you'll lose that game regardless if you have 3 or more cards matching cards.
